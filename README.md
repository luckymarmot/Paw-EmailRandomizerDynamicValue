#Email Randomizer Dynamic Value (Paw Extension)

A [Paw Extension](http://luckymarmot.com/paw/extensions/) that generates random email addresses.

## Installation

```shell
npm install
cake build
cake install
```

## Development

During development, watch for changes:

```shell
cake watch
```
